# Politique de Sécurité

## Versions Supportées

Veuillez utiliser la version la plus récente de l'application pour bénéficier des derniers correctifs de sécurité.

| Version | Supporté          |
| ------- | ----------------- |
| 1.0.x   | ✅ Oui            |
| < 1.0   | ❌ Non            |

## Signaler une Faille de Sécurité

Nous prenons la sécurité de ce projet très au sérieux. Si vous découvrez une vulnérabilité, veuillez suivre les étapes suivantes :

1. **Ne pas créer d'issue publique** pour signaler une faille de sécurité.
2. Envoyez un e-mail détaillé à **mohameden19961@github.com** (ou contactez-moi via mon profil GitHub).
3. Décrivez la vulnérabilité et les étapes pour la reproduire.

Nous nous efforçons de répondre sous 48 heures et de proposer un correctif dans les plus brefs délais.

Merci de nous aider à maintenir ce projet sécurisé !
